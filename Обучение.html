Языки программирования

Здесь находятся языки, с которыми я сталкивался во время учебы.

Delphi (Pascal)

Pascal - первый мой язык программирования. Начал изучать еще в 7 классе, а на практике использовал диалект данного языка в RAD Studio - Delphi.

C++

C++ - первый язык, который мы начали изучать в колледже, еще на первом курсе. На нем была написана моя первая курсовая работа - ППС "Методы сортировки".

HTML

На HTML я написал много работ, но не могу назвать его самым любимым языком. Проектирование интерфейса и качественного дизайна не совсем мое, как я считаю.

C#

С# я узнал только на третьем курсе. Как по мне, он не сильно отличается от С++, однако имеет некоторые аспекты, которые делают его более удобным.

CSS

Познал CSS несколько месяцев назад. Его использование при написании сайтов значительно облегчило работу. Больше сказать про CSS мне нечего.

JAVA

Java - язык, который я начал изучать самостоятельно. Структура его не совсем удобная после С++ и C#, но я привык и быстро осознал плюсы работы с ним.
